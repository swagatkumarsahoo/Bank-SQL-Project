# Bank Database Management System (Oracle SQL & PL/SQL)

## Description
A banking database project built using Oracle SQL and PL/SQL.
Implements account management and transaction handling.

## Concepts Used
- DDL & DML
- Constraints & Foreign Keys
- Stored Procedures
- Triggers
- Transactions

## Features
- Deposit money
- Withdraw money with PIN validation
- Transaction history
- Trigger-based balance validation
- Transfer money between accounts
- Transaction logging for all operations


## How to Run
Execute SQL files in order using VS Code or Oracle SQL Developer.
